# Comments

The assembler supports code comments, so you can add context explaining your assembly code to your future self and others who might work on it.

Comments begin with the `#` character and continue until the end of the line. They are ignored by the assembler and have no effect on program behavior.

**Usage examples:**
```misa
# This is a comment.
add t0, t1  # Another comment.
```

You can select a code block and use the shortcut `Ctrl + K` to toggle comment.
