# Glossary

Common terms in this chapter, described in the context of the console:

**Directive**
A statement used to control the assembler, such as defining labels, data, and constants.

**Assembly**
A low-level programming language used to write programs targeting MISA.

**Assembler**
A program that converts assembly code into bytecode for the console's virtual machine.

**Assemble**
The process of converting assembly code into bytecode using the assembler.

**Assemble-time**
The phase during which the assembler converts assembly code into bytecode. Analogous to "compile-time".

**Runtime**
The phase during which bytecode is executed by the virtual machine.

**Bytecode**
The binary representation of a program produced by the assembler and executed by the virtual machine.
