# Bookmarks

## Shortcuts in the Sidebar

With bookmark directives, you can easily navigate your code by adding shortcuts to relevant locations in the Code Editor's bookmark sidebar.

---

## Bookmark

Defines a bookmark.

**Syntax:**
```
bmk <string_title>
```

---

## Sub-Bookmark

Defines a sub-bookmark.

**Syntax:**
```
sbmk <string_title>
```
