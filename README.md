# MISA C Compiler
A C89 compiler that targets the MISA/Mnemonimov VM architecture.

## AI Usage Disclaimer
While I wrote all the code myself, the Makefile, the github workflows, and the comments in the header files were generated by Claude.

## Building

Requires `clang`.

Compiling with `make` is easy:

```
make
```

Here is the exact clang command I use if you don't have make:

```
clang -Iinclude -D_CRT_SECURE_NO_WARNINGS -D_CRT_NONSTDC_NO_DEPRECATE src/token.c src/ast.c src/type.c src/symtab.c src/lexer.c src/parser.c src/sema.c src/codegen.c src/main.c -o misacc.exe
```

## Usage

```
misacc.exe <input.c> [-o output.misa]
```

- `<input.c>` — C89 source file to compile (required)
- `-o output.misa` — write output to a file instead of stdout

**Examples:**

```
misacc.exe program.c -o program.misa
misacc.exe program.c > program.misa
```
